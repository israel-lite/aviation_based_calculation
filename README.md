# Eazee TSD Calc

A modern, interactive web application for learning and practicing Time-Speed-Distance (TSD) problems used in aviation.

## 🚀 Features

### 🧮 TSD Calculator
- Calculate Distance, Speed, or Time with a dropdown selector
- Real-time calculations with aviation units (knots, nautical miles, hours)
- Step-by-step solution display
- Professional aviation formatting

### 📝 Aviation Quiz Mode
- Random TSD questions with realistic aviation scenarios
- Multiple choice answers (A, B, C, D)
- Score tracking and optional timer
- Detailed explanations for each answer

### 🎓 Learning Mode
- Beginner-friendly explanations
- Aviation-specific concepts (knots, nautical miles)
- Conversion references
- Formula reference guide

### 🔄 Conversion Tool
- Minutes ↔ Hours converter
- Knots ↔ km/h converter
- Real-time conversion as you type
- Clear aviation unit explanations

### 🏆 Pilot Challenge Mode
- 5-question challenge with increasing difficulty
- Rank system: Cadet → First Officer → Captain
- Streak tracking and accuracy metrics
- Aircraft-specific scenarios

### 📊 History Tracker
- Stores last 10 calculations
- Local storage persistence
- Timestamp tracking
- Easy reference for review

## 🎯 How to Use

### Calculator
1. Select what you want to calculate (Distance/Speed/Time)
2. Enter the known values
3. Click "Calculate" for instant results
4. View step-by-step solution

### Quiz Mode
1. Click "Start Quiz"
2. Enable timer for added challenge
3. Select your answer
4. Learn from detailed explanations

### Challenge Mode
1. Click "Start Challenge"
2. Answer 5 questions of increasing difficulty
3. Earn your pilot rank
4. Try again to improve your score

## 🧮 Aviation Units Reference

- **1 Knot** = 1 nautical mile per hour
- **1 Nautical Mile** = 1.852 km = 1.151 miles
- **Speed Range**: 100-500 knots (typical aircraft speeds)
- **Time Format**: Decimal hours (e.g., 2.5 hours = 2 hours 30 minutes)
